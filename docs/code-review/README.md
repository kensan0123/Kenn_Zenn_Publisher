## 🎯 使い方

### レビュー結果を確認する
1. [2025-12-24-codebase-review.md](./2025-12-24-codebase-review.md)を開く
2. 優先度の高い問題から確認
3. 各問題の「なぜ問題か」を理解する

### タスクを進める
1. [action-items.md](./action-items.md)を開く
2. Phase 1のタスクから着手
3. 完了したタスクにチェックを入れる
4. 進捗率を更新

### 新しいレビューを追加する
```bash
# 新しいレビューファイルを作成
touch docs/code-review/YYYY-MM-DD-<topic>-review.md
```
